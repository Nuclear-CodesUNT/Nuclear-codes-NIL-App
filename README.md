# Nuclear-codes-NIL-App
App to support and streamline NIL deals to college athletes

## Prereqs 
1. Node.js (v16 or later)
2. npm (comes with node)

## Installation
```
git clone git@github.com:BM-UNT/Nuclear-codes-NIL-App.git
cd Nuclear-codes-NIL-App
```
### Install dependencies
```
cd backend
npm install

cd ../frontend
npm install
```
## Launch
Need two seperate terminal windows
```
cd backend
npm start
```
```
cd frontend
npm start
```
